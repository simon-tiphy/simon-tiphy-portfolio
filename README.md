- **Frontend**: React, Vite

## Getting Started

To run this project locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/simon-tiphy/simontiphyportfolio.git
   cd simontiphyportfolio
   ```

2. **Install Dependencies**

   ```bash
   cd frontend
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```
